Member Registration is a membership portal solution designed to provide a pixel perfect, responsive, membership marketing and registration experience.

Membership tiers, registrations, and members are all managed Salesforce. Users can authenticate using their email or social login of choice, select a membership tier, and sign up directly for a membership using this app. While all data ultimately resides in Salesforce, this app illustrates data being published as content to the app inside of Salesforce, along with the ability to capture end user registrations and insert them back into Salesforce via Heroku Connect. It also illustrates the ability to relate newly created registrations with the appropriate membership tiers and other related object through the use of External Id's within Heroku Connect.

The Member Registration app is built using Ruby on Rails. Salesforce data is synchronized bi-directionally with a Postgres DB using Heroku Connect.
